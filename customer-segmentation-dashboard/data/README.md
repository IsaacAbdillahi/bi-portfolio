# 📄 Data Overview

This folder contains the customer‑level transactional sales dataset used for the **Customer Segmentation & Value Analysis** project.

The data represents historical sales activity at an individual customer level and is used to analyse revenue concentration, customer value distribution, and value‑based customer segmentation.

---

## 📑 Dataset Description

The dataset includes the following fields:

- **customer_id** — Unique identifier for each customer  
- **order_date** — Date the transaction occurred  
- **sales_amount** — Revenue value of the transaction  
- **region** — Customer or transaction region  

Each row represents a single customer transaction.

---

## 🔍 Data Usage in This Project

The dataset is used to:

- Aggregate total revenue by customer
- Rank customers based on revenue contribution
- Analyse how revenue is distributed across the customer base
- Create value‑based customer segments
- Support visual exploration and insight communication in Power BI

Data is cleaned, aggregated, and segmented using SQL before being visualised in Power BI.

---

## ⚠️ Data Notes

- The dataset is **synthetic or publicly available** and used solely for portfolio demonstration purposes  
- No real customer, personal, or company data is included  
- The structure reflects realistic customer‑level transaction data commonly used in Business Intelligence and analytics workflows
