# 📄 Data Overview

This folder contains the transactional sales dataset used for the **Sales Performance Dashboard** project.

The data represents historical sales activity and is used to analyse revenue trends, product performance, regional performance, and time‑based growth.

---

## 📑 Dataset Description

The dataset includes the following fields:

- **order_date** — Date the sale was recorded  
- **product** — Product sold  
- **region** — Sales region  
- **sales_amount** — Revenue value of the transaction  

Each row represents a single sales transaction.

---

## 🔍 Data Usage in This Project

The dataset is used to:

- Aggregate sales performance over time
- Analyse month‑over‑month revenue growth
- Compare product and regional performance
- Support KPI reporting and dashboard visualisations in Power BI

Data is cleaned and transformed using SQL before being modelled and visualised in Power BI.

---

## ⚠️ Data Notes

- The dataset is **synthetic or publicly available** and used for portfolio demonstration purposes  
- No real customer or company data is included  
- Structure reflects real‑world transactional sales data commonly used in BI reporting
